# Setup Antd on Vue3

- yarn add ant-design-vue @ant-design/icons-vue
- Edit main.ts
- Check the example from https://antdv.com/components/button in App.vue


# Setup tailwindcss
- https://tailwindcss.com/docs/guides/vite#vue - disable preflight in tailwind.config.js

```js
  corePlugins: {
    preflight: false,
  },
```
